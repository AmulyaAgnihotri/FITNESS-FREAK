# 💪 Fitness Freak — Premium Gym Website

> **Fitness Freak Powered by Fitline** — Best Gym in Lakhimpur Kheri

A fully static, premium gym website with glassmorphic UI, Vimeo video embeds, BMI calculator, interactive schedule, and WhatsApp integration.

## 🌐 Live Features

- Responsive design (desktop, tablet, mobile)
- Vimeo-hosted video reels (transformations, trainers, gym tour)
- Interactive BMI calculator
- Weekly class schedule with accordion UI
- Testimonials carousel
- Google Maps embed
- WhatsApp direct-message CTAs
- Instagram profile integration

## 🚀 Deploy

This is a static site — no build step needed. Deploy directly to:

- **GitHub Pages** → Settings → Pages → Source: `main` branch, `/root`
- **Vercel** → Import repo → Framework: Other → Deploy
- **Netlify** → Drag-and-drop the project folder

## 🧑‍💻 Run Locally

```bash
# Any static server works:
npx serve .
# or
python -m http.server 4173
```
